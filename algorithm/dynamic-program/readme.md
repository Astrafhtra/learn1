一个草原有五百只狮子，一只羊，狮子吃羊？
狮子和羊都吃草生活， 狮子可以吃羊 一整只，这只狮子变成羊；

1. 不吃，（吃了会变成羊）
    当只有一只狮子，吃
2. 吃，羊肉好吃
    有两只狮子一只羊
3. 不吃，羊那么肥，做个朋友吧
  三只狮子，一只羊，吃
4. 四只狮子 ，不吃 
5. 五只狮子 ，吃

五百只狮子，不吃羊

大事情难解决，分成小事情 大事化小（递归）
f(n)=!f(n-1)
边界 n=1 吃；n=2 不吃
n=500 不吃
